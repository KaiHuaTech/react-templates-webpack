## 开始
```
$ npm install                   # Install project dependencies
$ npm start                     # Compile and launch
```


## 其他命令
| `npm run <script>` | 说明 |
| ---- | ---- |
| `start` | 启动开发服务器，开启HMR |
| `dev` | 与 `npm start` 相同 |
| `build` | 打包项目文件 |
| `lint` | 检查 `src` 下文件是否符合规范 |
| `lint：fix` | 检查 `src` 下文件是否符合规范并修复 |
